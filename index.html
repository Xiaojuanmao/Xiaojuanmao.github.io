Ugly Number

题目

Ugly Number

Ugly number is a number that only have factors 3, 5 and 7.

Design an algorithm to find the Kth ugly number. The first 5 ugly numbers are 3, 5, 7, 9, 15 ...

例子

If K=4, return 9.

解答

方法一 ： 暴力法

    对于这种问题，一开始拿到手上基本没啥想法= =，开始就从3开始往后检查，来一个算一个，对每个数进行检查，算完为止。不需要每个数都检察，很明显偶数是不可能是丑数Orz。(突然就感觉连数的世界也变的不公平了。

class Solution {
    /**
     * @param k: The number k.
     * @return: The kth prime number as description.
     */
    public long kthPrimeNumber(int k) {
        // write your code here

        if (k <= 0) return -1;

        int current = 1;

        while(k > 0) {
            current += 2;
            if (isUgly(current)) k--;
        }

        return current;
    }

    private boolean isUgly(int num) {

        while(num % 3 == 0) {
            num /= 3;
        }

        while(num % 5 == 0) {
            num /= 5;
        }

        while(num % 7 == 0) {
            num /= 7;
        }
        return 1 == num;
    }
};

方法二 ： 二分查找法

    上面那种方法很明显好多个循环，时间复杂度绝对直飙，考虑下面这种思路，也是挨个的寻找ugly number，假设现在已经找出了n个ugly number,需要寻找第n+1个数，根据规定只能有3,5,7这几个约数，所以肯定是从前面的n个数中找一个基础数，乘以3或者5或者7，使这个结果比第n个ugly number大，但是又是在前面n个数计算的结果中是最小的。

class Solution {
    /**
     * @param k: The number k.
     * @return: The kth prime number as description.
     */
    public long kthPrimeNumber(int k) {
        // write your code here

        if (k <= 0) return -1;
        ArrayList<Long> nums = new ArrayList();
        nums.add(1L);
        while(k > 0) {
            long nextUgly = Math.min(nextUglyNumber(nums, 3), nextUglyNumber(nums, 5));
            nextUgly = Math.min(nextUglyNumber(nums, 7), nextUgly);
            nums.add(nextUgly);
            k--;
        }
        return nums.get(nums.size() - 1);
    }

    private long nextUglyNumber(ArrayList<Long> nums, int factor) {
        int size = nums.size();
        int start = 0;
        int end = size - 1;

        while (start + 1 < end) {

            int mid = (start + end) / 2;

            long delta = nums.get(mid) * factor - nums.get(size - 1);

            if (delta <= 0) {
                start = mid;
            } else {
                end = mid;
            }
        }

        if    (nums.get(start)*factor>nums.get(size-1))    {
            return    nums.get(start)    * factor;
        }
        return    nums.get(end) *    factor;
    }
};

方法三 ： 动态规划

    这个方法也需要将前面计算出来的ugly number存储起来，类似于merge sort的方法，将符合条件的最小ugly number加入到存储的列表中去。

3 * uglyIndexThree
5 * uglyIndexFive
7 * uglyIndexSeven

    这个方法将ugly number的产生分成了三种类型，第n个ugly number的产生肯定是从上面这三种类型中选择，上面的三种uglyIndex是已经获取到的ugly number列表中的下标值。下标值从0开始，每次从这三类数中选取最小的那个ugly number,之后下标右移一个，如果有相同的ugly number,则下标同时移动一次，知道找到第n+1个ugly number。下面是Java的实现代码：

class Solution {
    /**
     * @param k: The number k.
     * @return: The kth prime number as description.
     */
    public long kthPrimeNumber(int k) {
        // write your code here

        if (k <= 0) return -1;
        int indexThree = 0;
        int indexFive = 0;
        int indexSeven = 0;

        ArrayList<Long> nums = new ArrayList();
        nums.add(3L);
        nums.add(5L);
        nums.add(7L);

        if (k - 3 < 0) return nums.get(k);
        k -= 3;
        while(k > 0) {

            long nextUgly = minNum(nums.get(indexThree) * 3, nums.get(indexFive) * 5, nums.get(indexSeven) * 7);
            if (nextUgly == nums.get(indexThree) * 3) {
                indexThree++;
            }
            if (nextUgly == nums.get(indexFive) * 5) {
                indexFive++;
            }
            if (nextUgly == nums.get(indexSeven) * 7) {
                indexSeven++;
            }
            nums.add(nextUgly);
            k--;
        }
        return nums.get(nums.size() - 1);
    }

    private long minNum(long a, long b, long c) {
        if (a > b) {
            if (b > c) {
                return c;
            } else  {
                return b;
            }
        } else {
            if (a > c) {
                return c;
            } else {
                return a;
            }
        }
    }

};
