关于SpannableString

前言

    最近在项目中接触到了聊天的模块，在聊天模块中处理了一些关于字符串的功能，例如将一串链接格式的字符串识别成带有下划线样式的链接字样，将预先规范好的字符串翻译成对应的表情字符，将带有特殊意义的字符串标识成不同颜色等。之前没有接触过，学习了一下相关知识，找了个时间稍微整理一下。

SpannableString

public class SpannableString
extends Object implements CharSequence, GetChars, Spannable

java.lang.Object
↳ android.text.SpannableString

This is the class for text whose content is immutable but to which markup objects can be attached and detached.For mutable text, see SpannableStringBuilder.

    官方对其解释为，为内容可变的文本进行标记性装饰的类，和SpannableStringBuilder结合起来使用。SpannableStringBuilder也就是用来方便构造各式各样Span的工具类，Android内部提供了很多种样式，部分Span大致使用方法如下：

    使用起来还是挺方便的，由于SpannableStringBuilder和SpannableString都继承自CharSequence,能够直接往TextView上面放。上面只是在字符串中截取了一段添加了下划线样式，Android还提供了各种花式的效果供选择：

DynamicDrawableSpan.ALIGN_BOTTOM

    ImageSpan继承自DynamicDrawableSpan类，上面的构造函数将verticalAlignment参数传给了父类处理，在DynamicDrawableSpan中看到了该参数发挥作用的地方，draw在绘制图片的时候会被调用，这时候会根据所指定的位置来画，基本上是想画到哪里就画到哪里。：
