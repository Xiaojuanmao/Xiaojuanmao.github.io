Super Ugly Number

313. Super Ugly Number

Write a program to find the nth super ugly number.

Super ugly numbers are positive numbers whose all prime factors are in the given prime list primes of size k. For example, [1, 2, 4, 7, 8, 13, 14, 16, 19, 26, 28, 32] is the sequence of the first 12 super ugly numbers given primes = [2, 7, 13, 19] of size 4.

Note:

  • 1 is a super ugly number for any given primes.
  • The given numbers in primes are in ascending order.
  • 0 < k ≤ 100, 0 < n ≤ 106, 0 < primes[i] < 1000.

思路

这个题目相当于是将因式分解的过程反过来了，通过给定的一个数组里的元素，组合出由小到大的公倍数，可以将n个Ugly Number都放在一个数组里面，算一个往数组里面放一个，最后将数组的最后一个元素取出来就是结果了，这个数组里面都是之前算出来的Ugly Number，可以利用之前的结果来避免重复的计算，只要每次将结果数组里面的数和primes数组里面的数相乘，得出一个刚好比Ugly Number数组中最大的数大的结果，加入其中即可。

这里能够有一个优化，对Ugly Number中每一个数字与primes中相乘的流程记录下来，记录最后一次相乘的index，利用这样一个过程就不用每次在试探的时候都去从primes的0号位开始乘，而是从上一条记录开始，这也是利用了从小到大的有序性做到了这一点。
