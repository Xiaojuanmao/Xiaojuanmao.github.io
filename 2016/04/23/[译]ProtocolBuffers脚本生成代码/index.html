  1. protoc : 脚本命令，类似于java或javac一样

    Message接口中提供一些方法给用户，用来检查、操作、读写实体message类，大概有下面这些方法：

  • static Foo getDefaultInstance() : 该方法返回一个Foo类的单例
  • static Descriptor getDescriptor() : 返回对类型的描述，描述包含这个message里面包含了一些信息的类型等等。
  • static Foo parseFrom(...) : 从给定的源数据中解析并返回一个Foo类型的message
  • static Parser parser() : 返回一个Parser的实例，该实例中包含这多种不同的parseFrom()方法。
  • Foo.Builder newBuilder() : 创建一个builder
  • Foo.Builder new Builder(Foo prototype) : 创建一个builder,builder里面的值根据传入的参数来初始化
